## Cara Menggunakan Prompt ChatGPT untuk UMKM

Prompt ChatGPT untuk UMKM akan lebih bermanfaat jika langsung dikaitkan dengan pekerjaan bisnis sehari-hari.

Jangan hanya memakai prompt umum seperti:

```text
Buatkan ide bisnis.
```

Prompt seperti itu terlalu luas.

Untuk UMKM, prompt yang lebih baik harus menyebutkan:

- jenis usaha,
- produk atau jasa,
- target pelanggan,
- masalah yang ingin diselesaikan,
- channel jualan,
- gaya bahasa,
- format output,
- tujuan akhir.

Contoh prompt yang lebih jelas:

```text
Bertindaklah sebagai konsultan bisnis UMKM.

Saya punya usaha:
[jelaskan usaha]

Produk/jasa:
[jelaskan produk atau jasa]

Target pelanggan:
[jelaskan target]

Channel jualan:
[Instagram/WhatsApp/marketplace/toko offline/website]

Masalah utama:
[jelaskan masalah]

Bantu saya membuat:
1. ide promosi
2. caption
3. template WhatsApp
4. ide konten
5. cara follow-up pelanggan
6. checklist yang bisa langsung saya pakai

Gunakan bahasa Indonesia sederhana dan praktis.
```

Dengan konteks seperti ini, hasil ChatGPT akan lebih relevan untuk bisnis kecil.

## Prompt ChatGPT untuk Promosi UMKM

Promosi adalah kebutuhan paling umum untuk UMKM.

ChatGPT bisa membantu membuat ide promosi yang lebih variatif.

Gunakan prompt berikut:

```text
Bertindaklah sebagai marketing strategist untuk UMKM.

Saya memiliki usaha:
[jenis usaha]

Produk utama:
[nama produk]

Target pelanggan:
[target pelanggan]

Harga produk:
[harga]

Channel promosi:
[Instagram/WhatsApp/TikTok/Facebook/marketplace]

Buatkan:
1. 10 ide promosi
2. 5 ide promo bundling
3. 5 ide promo terbatas
4. 5 caption promosi
5. 5 CTA
6. ide visual untuk setiap promosi

Batasan:
Jangan membuat klaim berlebihan.
Gunakan bahasa yang ramah dan mudah dipahami.
```

Contoh penggunaan:

```text
Usaha:
kopi susu rumahan

Target:
mahasiswa dan pekerja kantor sekitar area kampus

Channel:
Instagram dan WhatsApp

Tujuan:
meningkatkan pesanan harian
```

Prompt ini bisa membantu UMKM membuat promosi tanpa harus mulai dari nol.

```text

→ catat produk terlaris

```

## Kesalahan Menggunakan ChatGPT untuk UMKM

## 1. Prompt Terlalu Umum

Jika prompt terlalu umum, hasilnya juga umum.

Selalu beri konteks usaha, produk, pelanggan, dan tujuan.

## 2. Semua Output Langsung Dipakai

Output ChatGPT tetap harus diedit.

Sesuaikan dengan stok, harga, promo, aturan bisnis, dan gaya bahasa brand.

## 3. Membuat Klaim Berlebihan

Hindari klaim seperti:

```text
pasti laris,
pasti viral,
terbaik nomor satu,
paling ampuh,
dijamin semua orang suka.
```

Gunakan kalimat yang lebih aman dan realistis.

## 4. Tidak Mencatat Hasil

UMKM perlu mencatat apakah promosi berhasil.

Minimal catat:

```text
konten yang diposting,
jumlah chat masuk,
jumlah order,
produk terjual,
komentar pelanggan,
promo yang dipakai.
```

## 5. Tidak Menjaga Data Pelanggan

Jangan memasukkan data pribadi pelanggan ke tools AI publik.

Hindari memasukkan:

- nomor telepon,
- alamat lengkap,
- bukti transfer,
- data identitas,
- informasi pribadi sensitif.

Channel jualan disebutkan
