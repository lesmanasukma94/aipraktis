## Cara Menggunakan Prompt ChatGPT untuk Sales

Prompt ChatGPT untuk sales sebaiknya tidak digunakan dengan cara asal copy-paste.

Agar hasilnya lebih berguna, Anda perlu memberi konteks yang jelas.

ChatGPT perlu tahu:

- produk atau jasa yang dijual,
- target pelanggan,
- masalah pelanggan,
- harga atau paket,
- gaya komunikasi brand,
- channel penjualan,
- tahap funnel,
- tujuan pesan,
- batasan yang tidak boleh dilanggar.

Contoh prompt yang terlalu umum:

```text
Buatkan pesan sales.
```

Prompt seperti itu biasanya menghasilkan jawaban yang terlalu generik.

Prompt yang lebih baik:

```text
Bertindaklah sebagai sales copywriter.

Saya menjual:
[jelaskan produk/jasa]

Target pelanggan:
[jelaskan target pelanggan]

Masalah utama pelanggan:
[jelaskan masalah]

Tujuan pesan:
[mengajak konsultasi / follow-up / closing / menjawab keberatan]

Channel:
[WhatsApp / email / DM Instagram / LinkedIn]

Buatkan pesan sales yang:
- singkat
- sopan
- tidak terlalu memaksa
- fokus pada manfaat
- punya CTA jelas
```

Dengan konteks seperti ini, hasil ChatGPT akan lebih relevan.

## Kategori Prompt ChatGPT untuk Sales

Prompt sales sebaiknya dibagi berdasarkan tahap penjualan.

Jangan memakai satu jenis prompt untuk semua situasi.

Kebutuhan sales saat mencari prospek berbeda dengan saat follow-up.

Kebutuhan saat menjawab keberatan juga berbeda dengan saat closing.

Berikut kategori prompt yang paling berguna.

## 1. Prompt untuk Memahami Target Pelanggan

Sebelum menjual, pahami dulu siapa calon pelanggan.

Gunakan prompt seperti ini:

```text
Bertindaklah sebagai sales strategist.

Saya menjual:
[produk/jasa]

Target pelanggan sementara:
[target]

Bantu saya menganalisis:
1. masalah utama calon pelanggan
2. kebutuhan yang paling mendesak
3. alasan mereka butuh solusi ini
4. keberatan yang mungkin muncul
5. bahasa yang cocok untuk mendekati mereka
6. channel penjualan yang paling relevan
7. sudut penawaran yang paling kuat

Gunakan bahasa Indonesia praktis.
```

Prompt ini berguna sebelum membuat pesan penjualan.

## 2. Prompt untuk Membuat Pesan Pertama

Pesan pertama harus sopan dan tidak terlalu agresif.

Contoh prompt:

```text
Buatkan 5 variasi pesan pendek untuk menghubungi prospek pertama kali.

Produk/jasa:
[jelaskan]

Target prospek:
[jelaskan]

Konteks:
[saya belum kenal / sudah pernah bertemu / pernah follow akun / pernah bertanya]

Channel:
[WhatsApp / DM Instagram / email / LinkedIn]

Batasan:
- jangan terlalu panjang
- jangan hard selling
- fokus pada masalah prospek
- akhiri dengan pertanyaan ringan
```

Contoh CTA yang lebih natural:

```text
Apakah bagian ini sedang jadi prioritas di bisnis Anda?
```

Atau:

```text
Boleh saya kirim contoh solusinya?
```

## 3. Prompt untuk Follow-Up Prospek

Banyak penjualan gagal karena follow-up tidak rapi.

ChatGPT bisa membantu membuat variasi follow-up agar tidak terdengar seperti spam.

Contoh prompt:

```text
Buatkan 7 pesan follow-up untuk prospek.

Konteks:
Prospek sudah tertarik, tetapi belum memberi keputusan.

Produk/jasa:
[jelaskan]

Keberatan sementara:
[belum ada budget / masih pikir-pikir / perlu diskusi dengan tim]

Gaya:
sopan, ringan, membantu, tidak memaksa.

Format:
hari ke-1 sampai hari ke-7.
Setiap pesan maksimal 3 kalimat.
```

Follow-up yang baik tidak hanya bertanya:

```text
Jadi ambil atau tidak?
```

Follow-up yang lebih baik memberi nilai tambahan.

Misalnya:

- mengirim contoh,
- menjawab keberatan,
- memberi studi kasus,
- memberi checklist,
- menawarkan diskusi singkat,
- mengingatkan deadline promo jika memang ada.

## 4. Prompt untuk Menjawab Keberatan

Dalam sales, keberatan adalah hal normal.

Contoh keberatan:

- harga terlalu mahal,
- belum butuh,
- sudah punya vendor,
- perlu pikir-pikir,
- belum ada budget,
- takut hasilnya tidak sesuai,
- belum percaya,
- harus diskusi dengan partner,
- belum yakin manfaatnya.

Gunakan prompt:

```text
Bertindaklah sebagai sales consultant.

Saya ingin menjawab keberatan calon pelanggan.

Produk/jasa:
[jelaskan]

Keberatan pelanggan:
[tempel keberatan]

Profil pelanggan:
[jelaskan singkat]

Buatkan:
1. jawaban singkat
2. jawaban lebih lengkap
3. versi WhatsApp
4. versi email
5. pertanyaan lanjutan untuk memahami kebutuhan mereka

Batasan:
Jangan memaksa.
Jangan merendahkan pelanggan.
Fokus pada manfaat dan kejelasan.
```

Jawaban keberatan harus terasa membantu, bukan menyerang.

## 5. Prompt untuk Closing

Closing bukan berarti menekan pelanggan.

Closing yang baik membantu pelanggan mengambil keputusan dengan jelas.

Contoh prompt:

```text
Buatkan 5 variasi pesan closing.

Konteks:
Prospek sudah memahami penawaran dan tertarik.

Produk/jasa:
[jelaskan]

Paket:
[jelaskan paket]

Harga:
[harga]

Bonus atau batas waktu:
[jelaskan jika ada]

Gaya:
sopan, jelas, profesional, tidak memaksa.

CTA:
ajak memilih paket atau menjadwalkan pembayaran.
```

Contoh CTA closing:

```text
Kalau sudah cocok, saya bisa bantu siapkan invoice dan jadwal pengerjaannya.
```

Atau:

```text
Apakah Bapak/Ibu ingin mulai dari paket basic dulu agar risikonya lebih kecil?
```

## Contoh Workflow Sales dengan ChatGPT

Berikut workflow sederhana memakai ChatGPT untuk membantu proses sales.

```text
Riset prospek
→ pahami masalah
→ buat pesan pertama
→ kirim penawaran
→ follow-up
→ jawab keberatan
→ closing
→ onboarding pelanggan
→ after-sales
```

## 1. Riset Prospek

Gunakan ChatGPT untuk membuat daftar hal yang perlu diperhatikan.

Contoh prompt:

```text
Saya ingin mendekati calon pelanggan dengan profil berikut:

Profil:
[jelaskan prospek]

Bisnis:
[jelaskan bisnis]

Produk/jasa saya:
[jelaskan]

Bantu saya membuat:
- kemungkinan masalah mereka
- kebutuhan yang mungkin relevan
- sudut pendekatan
- pesan pembuka
- pertanyaan discovery
```

## 2. Discovery Call

Sebelum menjual, ajukan pertanyaan.

Contoh prompt:

```text
Buatkan 15 pertanyaan discovery untuk calon pelanggan.

Produk/jasa:
[jelaskan]

Target pelanggan:
[jelaskan]

Tujuan:
memahami kebutuhan, masalah, budget, timeline, dan pengambil keputusan.

Buat pertanyaan yang sopan dan tidak terasa menginterogasi.
```

Contoh pertanyaan:

```text
Saat ini bagian mana yang paling sering menghambat proses penjualan?
```

```text
Apa yang sudah pernah dicoba sebelumnya?
```

```text
Kalau solusi ini berhasil, hasil seperti apa yang paling penting untuk Anda?
```

## 3. Penawaran

Setelah memahami kebutuhan, baru buat penawaran.

Contoh prompt:

```text
Buatkan draft penawaran sales berdasarkan informasi berikut.

Produk/jasa:
[jelaskan]

Masalah pelanggan:
[jelaskan]

Solusi:
[jelaskan]

Output:
[jelaskan deliverables]

Harga:
[jelaskan]

Timeline:
[jelaskan]

Buat format:
1. pembuka
2. ringkasan masalah
3. solusi
4. output
5. timeline
6. harga
7. CTA
```

## 4. Follow-Up

Buat jadwal follow-up agar tidak lupa.

Contoh prompt:

```text
Buatkan sequence follow-up 10 hari untuk prospek.

Kondisi:
Prospek sudah menerima penawaran tetapi belum memberi keputusan.

Buat:
- pesan hari ke-1
- pesan hari ke-3
- pesan hari ke-5
- pesan hari ke-7
- pesan hari ke-10

Setiap pesan harus memberi nilai, bukan hanya menagih keputusan.
```

## 5. After-Sales

Sales tidak selesai setelah pembayaran.

Gunakan ChatGPT untuk membuat komunikasi setelah pelanggan membeli.

Contoh prompt:

```text
Buatkan pesan onboarding untuk pelanggan baru.

Produk/jasa:
[jelaskan]

Hal yang perlu pelanggan siapkan:
[daftar kebutuhan]

Timeline:
[jelaskan]

Buat pesan yang:
- ramah
- jelas
- memberi ekspektasi
- menjelaskan langkah berikutnya
```

After-sales yang baik bisa meningkatkan repeat order dan referral.

## Prompt ChatGPT untuk Sales WhatsApp

WhatsApp sering dipakai untuk sales di Indonesia.

Pesan WhatsApp harus singkat dan jelas.

Jangan terlalu formal.

Contoh prompt:

```text
Buatkan pesan WhatsApp untuk follow-up prospek.

Konteks:
[jelaskan situasi]

Produk/jasa:
[jelaskan]

Tahap prospek:
[baru tanya / sudah minta harga / sudah diberi penawaran / belum membalas]

Gaya:
ramah, sopan, tidak memaksa.

Batasan:
Maksimal 4 kalimat.
Akhiri dengan pertanyaan ringan.
```

Contoh hasil yang diharapkan:

```text
Halo Kak, saya follow-up singkat ya.

Kemarin Kakak sempat tanya tentang paket [nama paket]. Kalau masih dipertimbangkan, saya bisa bantu jelaskan paket yang paling sesuai dengan kebutuhan Kakak dulu.

Saat ini yang paling dibutuhkan lebih ke [opsi A] atau [opsi B]?
```

Untuk bisnis B2B, gunakan gaya lebih profesional.

```text
Halo Pak/Bu, saya follow-up terkait penawaran yang kemarin saya kirim.

Jika berkenan, saya bisa bantu ringkas opsi yang paling relevan agar lebih mudah didiskusikan dengan tim.

Apakah minggu ini masih menjadi waktu yang tepat untuk membahasnya?
```

## Prompt ChatGPT untuk Objection Handling

Gunakan template ini untuk membuat jawaban keberatan.

```text
Saya ingin menjawab keberatan prospek.

Produk/jasa:
[jelaskan]

Keberatan:
[tempel kalimat prospek]

Konteks:
[jelaskan tahap komunikasi]

Buatkan jawaban dalam 3 versi:
1. sangat singkat untuk WhatsApp
2. profesional untuk email
3. konsultatif untuk telepon

Batasan:
Jangan memaksa.
Jangan defensif.
Akui kekhawatiran pelanggan.
Berikan alasan yang masuk akal.
Akhiri dengan pertanyaan lanjutan.
```

Contoh keberatan:

```text
Harganya masih terlalu mahal.
```

Jawaban yang baik tidak langsung memberi diskon.

Coba pahami dulu.

```text
Saya paham, Pak/Bu. Supaya lebih sesuai, boleh saya tahu bagian mana yang terasa paling berat: total biayanya, timing pembayarannya, atau belum yakin dengan hasil yang akan didapat?
```

Pertanyaan seperti ini membuka percakapan, bukan menutupnya.

## Kesalahan Menggunakan ChatGPT untuk Sales

## 1. Mengirim Output AI Mentah

Pesan AI sering terdengar terlalu rapi atau terlalu promosi.

Edit sebelum dikirim.

Tambahkan konteks manusia.

## 2. Terlalu Banyak Kata-Kata Promosi

Hindari kalimat yang terlalu bombastis.

Contoh:

```text
Solusi terbaik dan paling revolusioner untuk bisnis Anda.
```

Lebih baik:

```text
Solusi ini membantu merapikan proses follow-up agar prospek tidak mudah terlewat.
```

## 3. Tidak Menyesuaikan Channel

Pesan WhatsApp, email, DM, dan LinkedIn tidak sama.

WhatsApp lebih singkat.

Email lebih rapi.

LinkedIn lebih profesional.

DM Instagram lebih ringan.

## 4. Tidak Mengecek Klaim

Jangan membuat klaim seperti:

```text
Pasti omzet naik 10x.
```

Gunakan klaim yang lebih aman:

```text
Dapat membantu mempercepat proses follow-up dan membuat komunikasi sales lebih konsisten.
```

## 5. Tidak Menyimpan Template yang Berhasil

Jika ada prompt atau pesan yang berhasil, simpan sebagai template.

Buat folder sederhana:

```text
sales/
  discovery/
  follow-up/
  objection/
  closing/
  onboarding/
```

Dengan begitu, proses sales berikutnya lebih cepat.
