# 1. Tujuan Dokumen

Dokumen ini mendefinisikan struktur seluruh halaman AI Praktis Indonesia.

Sitemap dirancang agar:

* mudah dipahami pengguna;
* mudah dijelajahi mesin pencari;
* mudah dikembangkan;
* mendukung ribuan halaman di masa depan.

---

# 2. Prinsip Sitemap

Seluruh sitemap mengikuti prinsip berikut.

* Flat Architecture
* SEO First
* User First
* Scalable
* Semantic
* Mobile First

Target utama adalah membuat pengguna dapat mencapai halaman yang diinginkan dalam maksimal tiga klik.

# 4. Homepage

```text
/
```

Homepage menjadi pusat navigasi seluruh website.

Fungsi:

* memperkenalkan AI Praktis Indonesia;
* mengarahkan pengguna ke pilar utama;
* menampilkan konten terbaru;
* menampilkan AI Tools populer;
* menampilkan Prompt populer.
